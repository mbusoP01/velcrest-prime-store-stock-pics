# Velcrest Prime — product photos

Public image host for the Velcrest Prime Shopify store's product photos, referenced by the
import CSVs in the (private) main repo. Contains photos only — no pricing or business data.

- `images/` — 492 photos for the VEL300 pool-equipment catalog, named by SKU (e.g. `630-3030.jpg`)
- `images_bags/` — 25 photos for the Cadii/Todii school bag colour variants

Regenerated from the main repo's `data/output/` folders. Do not edit here directly.
